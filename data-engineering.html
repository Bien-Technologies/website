Build dot com

Technology Used: UiPath
Target Systems: Oracle EBS, PO Console(web application), MS Excel and MS Outlook

Overview:
Automated process read PO (Purchase order number) from particular email subject. After reading the PO from email robot creates PO into Oracle EBS. Once order is created robot keeps track of some details into EBS once those are available robot login into PO console and update the details into PO console. After completion of process robot notify business users about success/failure of process.

Cash Reconciliation

Technology Used: UiPath
Target Systems: Oracle EBS, MS Excel and MS Outlook

Overview:
Automated solution perform Cash Reconciliation steps in Oracle EBS for particular statement date. First robot perform Load statement of particular date by executing particular responsibility into EBS then execute Auto reconcile responsibility. In end it perform Case reconciliation steps for remaining unreconciled items. After completion of process robot notify business users by email with attached detailed report of unreconciled items.

S2P

Technology Used: UiPath
Target Systems: CVENT (web portal), SmartBuy Ariba (web portal), MS Excel and MS Outlook

Overview:
This process automates the creation of purchase requisition (PR) in SmartBuy Ariba for each newly awarded Request for Proposal (RFP) in CVENT. Robot pick details of each awarded RFP from CVENT by traversing on multiple pages, consolidate the data fetched detail and save into excel. Then process each RFP one by one and create the PR in SmartBuy Ariba. If any business/system error occurred during the process the robot send email to individual/group. At the end of the process a report is generated from the saved excel and robot sends the report to business users.

JERE

Technology Used: UiPath
Target Systems: SAP, PDF, MS Excel, WPA (web portal) and MS Outlook

Overview:
Automated process fetch debit and credit detail from SAP by executing particular transaction code for predefined company code. Update that details in predefined excel and generate a text file by executing inbuilt macro of excel template. Then go to WPA site and navigate to particular company code to upload the text file. After uploading the text file robot execute same transaction code for same company code in SAP and check for the available value. Take the screen shot of SAP screen paste it into excel and generate a PDF file. Then go to WPA site again and upload the PDF file. On successful completion of process robot sends a email to business users.
